# waste-food-project

This repository contains the Proof of Concept for the Video Playback Control Service technical assessment. 

The application is a lightweight REST API built with Python and FastAPI, which directly controls a local VLC Media Player instance via OS commands and TCP sockets.

## Documentation & Instructions
Please refer to the **[Design_Document.pdf](./WasteFoodController.pdf)** included in this repository for:
- The complete system design and architecture
- Setup and run instructions
- API endpoints description
- Future improvements
